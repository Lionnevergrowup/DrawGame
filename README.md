# Bluey 画图填色 🎨

一个单页面的网页游戏,适合小朋友在 iPad 或浏览器里画画涂色。

## 怎么用

1. 把仓库克隆下来,直接双击 `index.html`,任何浏览器都能打开,不用装东西也不用启动服务器。
2. 第一次打开会自动弹出图文使用说明,看完点"明白了"就能开始玩。任何时候点顶部 **? 帮助** 都能再看。

> iPad 用 Safari 打开后,可以从"分享 → 添加到主屏幕",图标会出现在桌面,以后点开就是全屏 App。

## 功能

- **12 张内置图**:Bluey、Bingo、小狗屋、玩具球、爪印、蝴蝶、小鱼、彩虹、生日蛋糕、小花、小汽车、空白画板
- **加自己的图**:
  - 📁 从相册/文件上传(iPad 也能直接调相册)
  - 🔍 一键打开 Google 图片搜图,把喜欢的图存下来再上传
  - 直接粘贴图片网址(部分网站有跨站限制,推荐先下载再上传更稳)
- **三种工具**:点击填色 / 画笔 / 橡皮
- **15 色调色板 + HTML 颜色选择器**
- **画笔粗细可调 2–40 像素**,带实时预览
- **撤销 / 清空 / 保存**(保存会把所有涂色和笔迹合成一张 PNG 下载)
- **触屏优化**:iPad 上手指直接画,无双指缩放干扰
