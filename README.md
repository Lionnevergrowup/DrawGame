# 卡通画图填色 🎨

给小朋友的画图涂色单页 Web App。在电脑/iPad/手机浏览器里直接玩,所有数据保存在你
这台设备的浏览器里(localStorage),不上传任何地方。

线上地址 👉 **https://lionnevergrowup.github.io/DrawGame/**

## 功能

- **50 张内置图样**,按 🐾 动物 / 🌊 海洋 / 🦕 童话奇幻 / 🚗 交通工具 / 🌸 自然 / 🍰 食物 / 🎉 节日和其他 分类
- **30 种贴纸**(小人、动物、花草、星星爱心、太阳云彩…),选了在画上点一下就贴上
- **三种工具**:点击填色 / 自由画笔 / 橡皮
- **8 种填色纹路**:纯色、圆点、条纹、网格、锯齿、心、星、鱼鳞、波浪 —— 不只是单一颜色
- **20 色调色板 + 任意颜色选择器**
- **加自己的图**:从相册/文件上传,或一键跳到谷歌图片搜图后保存上传
- **放大缩小**:iPad 双指捏合,电脑用底部 +/− 按钮或 Ctrl+滚轮
- **撤销 / 清空 / 保存 PNG**(顶部大按钮,5 岁也能点)
- **自动保存**:每画一笔都本地存,关掉浏览器再打开还在;每张图各自存,换图不会丢
- **倒计时**:默认 10 分钟,可改 5/15/20/30/45/60 分,到点弹提示
- **全屏按钮**:点了浏览器进入全屏(iPad Safari 不支持全屏 API,推荐"分享 → 添加到主屏幕"装成 App)

## 开发(给开发者)

源码在 `build.py`(Python 构建脚本),浏览器加载的 `index.html` 是它生成的。

```bash
python3 build.py    # 重新生成 index.html
```

详细架构和约定见 `CLAUDE.md`。
