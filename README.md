# Bluey 画图填色 🎨

一个单页面的网页游戏,适合小朋友在 iPad 或浏览器里画画涂色。

## 玩法

直接在浏览器打开 `index.html` 即可,无需任何安装或服务器。

### 功能

- **点击填色**:选好颜色,点 Bluey 身体的任意区域就能上色
- **画笔**:在画面上自由涂鸦
- **橡皮**:擦掉画笔的笔迹
- **撤销 / 清空**:支持一步步撤销,或一键重置
- **保存**:把作品下载成 PNG 图片
- **多个图样**:Bluey、Bingo、小狗屋、玩具球、爪印、空白画板
- **15 色调色板 + 自选颜色**:满足各种创意
- **触屏优化**:在 iPad 上原生支持手指绘画,无双指缩放干扰

### iPad 使用建议

用 Safari 打开后,可以从"分享 → 添加到主屏幕"安装成全屏 Web App,体验更接近原生应用。
