# 卡通画图填色 🎨

给小朋友的画图涂色单页 Web App。在电脑/iPad/手机浏览器里直接玩,所有数据保存在你
这台设备的浏览器里(localStorage),不上传任何地方。

线上 👉 **https://lionnevergrowup.github.io/DrawGame/**

## 功能

- **100 张内置图样**,按 10 个分类:🐾 动物 / 🌊 海洋 / 🐦 鸟和昆虫 / 👦 小人 / 🦕 童话奇幻 / 🚗 交通工具 / 🏡 房屋建筑 / 🌸 自然 / 🍰 食物 / 🎉 节日和其他。每张拆成 15–30 个小区块,点一下不会一大片变色
- **30 种贴纸**(小人、动物、花草、星星爱心、太阳云彩…)。可调大小,可用纹路填充
- **8 种工具 + 多种填色纹路**:点击填色 / 自由画笔 / 橡皮 / 贴纸,共 20 种纹路(纯色 + 圆点 + 大圆点 + 斜条纹 + 横条纹 + 网格 + 格子 + 锯齿 + V形 + 心 + 星 + 三角形 + 砖块 + 十字 + 鱼鳞 + 波浪 + 泡泡 + 小花 + 豹纹 + 箭头)
- **35 色调色板 + 任意颜色选择器**,主屏只有"颜色"和"纹路"两个按钮,点开才出选项
- **加自己的图**:从相册/文件上传,或粘贴图片网址,或用 Google Custom Search API 直接搜图(需要你提供 API key,在弹窗里有详细引导,一次性 5 分钟设置好)
- **放大缩小拖动**:iPad 双指捏合,电脑用底部 +/− 按钮或 Ctrl+滚轮
- **撤销 / 清空 / 保存 PNG** 都在顶部大按钮
- **自动保存到 localStorage**:每画一笔都本地存,关掉再开还在。每张图分别保存,换图不会丢
- **倒计时**:支持**单人(默认 10 分钟)+ 多人轮流**(2–5 人,每人 1–15 分钟,时间到弹"换玩家")
- **全屏按钮**:浏览器进入全屏(iPad Safari 推荐用"分享 → 添加到主屏幕"装成 App)
- **响应式**:桌面 / iPad / iPhone 都适配,iPhone 上刘海区和底部安全区都正确处理

## 安全 / 隐私

- 所有数据(画作、自定义图、Google API key)只存在你这台设备的浏览器里,**不上传任何地方**
- 没有任何分析、追踪、广告
- 没有外部依赖,断网也能玩(除非用 Google 搜图功能)

## 开发(给开发者)

源码模块化拆成 3 个文件,**浏览器加载的 `index.html` 是它们生成的**:

```
build.py       — 主构建脚本(HTML/CSS/JS 模板 + 拼装逻辑)
templates.py   — 100 张图样的 SVG
stamps.py      — 30 个贴纸的 SVG
```

构建:

```bash
python3 build.py    # 重新生成 index.html
```

详细架构和约定见 `CLAUDE.md`。
