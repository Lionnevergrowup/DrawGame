# 卡通画图填色 🎨

给小朋友的画图涂色单页 Web App。在电脑 / iPad / 手机浏览器里直接玩,所有数据保
存在你这台设备的浏览器里(localStorage),不上传任何地方。

线上 👉 **https://lionnevergrowup.github.io/DrawGame/**

## 功能

- **100 张内置图样**,按 10 个分类整理:🐾 动物 / 🌊 海洋 / 🐦 鸟和昆虫 /
  👦 小人 / 🦕 童话奇幻 / 🚗 交通工具 / 🏡 房屋建筑 / 🌸 自然 / 🍰 食物 /
  🎉 节日和其他。每张图拆成 15–30 个小区块,点一下不会一大片变色
- **30 个贴纸**(小人、动物、花草、星星、爱心、太阳云朵…),大小可调,可用纹路填充
- **4 种工具**:点击填色 / 自由画笔 / 橡皮 / 贴纸
- **20 种纹路** + **35 色调色板 + 任意颜色选择器**。颜色和纹路收在 popup
  按钮里,主屏不挤
- **加自己的图**:从相册/文件上传,或粘贴图片网址,或用 Google Custom Search API
  搜图(需要自己提供 API key,弹窗里有引导)
- **放大缩小拖动**:iPad 双指捏合,电脑用底部 +/− 按钮或 Ctrl + 滚轮
- **撤销 / 清空 / 保存 PNG** 都在顶部大按钮
- **自动保存到 localStorage**:每画一笔本地存,关掉再开还在。每张图各自分开存
- **倒计时**:
  - **单人**:1 / 2 / 3 / 5 / 10 / 15 分钟,到时弹提示
  - **多人轮流**:2–5 人,每人 1–15 分钟,总共 1–10 轮(或不限)。换人时弹提示,
    最后一轮结束弹"都画完啦",可"再玩一局"
- **全屏按钮**:浏览器全屏(iPad Safari 推荐"分享 → 添加到主屏幕"装成 App)
- **响应式**:桌面 / iPad / iPhone 都适配,iPhone 上刘海区和底部安全区都处理好了

## 安全 / 隐私

- 所有数据(画作、自定义图、Google API key)只存在你这台设备的浏览器里,
  **不上传任何地方**
- 没有任何分析、追踪、广告
- 没有外部依赖,断网也能玩(除非用 Google 搜图)

## 开发

源码模块化拆成三个文件,**浏览器加载的 `index.html` 是它们生成的**:

| 文件 | 干什么 |
|---|---|
| `build.py` | 主构建脚本(HTML / CSS / JS 模板 + 拼装逻辑 + 配置常量) |
| `templates.py` | 100 张图样的 SVG 数据 |
| `stamps.py` | 30 个贴纸的 SVG 数据 |

构建:

```bash
python3 build.py    # 重新生成 index.html
```

详细架构、运行时模块速查、SVG 模板约定见 `CLAUDE.md`。
